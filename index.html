InfoCer

El proyecto se destinado para el sistema de certificación por evaluaciones parciales (EXACER) del Colegio de Bachilleres.


La aplicación se desarrolló como proyecto final, donde se buscó una solución ante la problemática que planteo Colegio de Bachilleres, que fue la siguiente.


En vista de las necesidades se ha propuesto el desarrollo de una aplicación web diseñada para visualizarse en dispositivos móviles cuyo objetivo principal será informar los datos básicos indispensables de una manera rápida en el momento que están en la sede para presentar el examen, además de encontrar información adicional como consulta de calificaciones, tramitación de certificados, generar reportes de insistencias he información relativa a las instalaciones, por ejemplo, servicio médico, salida de emergencia, cafetería, baños y permitir a los administradores (validadores, aplicadores, control de flujo, coordinadores de la sede) realizar reportes de incidencias dentro de los planteles que son asignados.


(No se uso framework ya que la universidad lo pidió como requisito del proyecto final, aun que el Colegio de Bachilleres si requería el uso framework como Bootstrap).
